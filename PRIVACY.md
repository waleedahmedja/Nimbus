# Privacy Policy

**Effective date:** March 2026
**Last updated:** March 2026

Nimbus is a weather app designed for personal use. This policy explains how information is — and is not — handled.

---

## Short Version

Nimbus stores your location locally to fetch weather. It stores your units preference locally. The daily sentence is generated by Claude API using anonymised weather data — no personal identifiers are sent. Everything else stays on your device.

---

## 1. Data Collection

Nimbus does not collect, transmit, or sell personal data.

The app does not collect:
- Personal identifiers of any kind
- Contact information
- Usage analytics or crash reports
- Advertising identifiers
- Biometric data
- Keystrokes or screen content

All app data (units preference, cached daily sentence, last known location) is stored locally on your device using Android DataStore. Uninstalling the app removes all of it.

---

## 2. Location

Nimbus requests device location to fetch a weather forecast for your current position.

Location is:
- Requested via Android's Fused Location Provider
- Used only to generate a latitude/longitude pair for the Open-Meteo API call
- Not stored beyond the session in any identifiable form
- Never transmitted to Nimbus servers — there are none

The app does not background-track your location. Location is fetched once when the app opens, and again when you manually refresh.

---

## 3. Weather Data

Forecasts are fetched from [Open-Meteo](https://open-meteo.com), a free, open-source weather API. Only a latitude and longitude are sent. No personal data is attached to this request. Open-Meteo's privacy policy governs their handling of request data.

---

## 4. Daily Sentence — Claude API

The daily AI sentence is generated by a single API call to Anthropic's Claude API (claude-haiku model).

The request contains:
- Current weather condition
- High and low temperatures
- Precipitation chance
- Wind speed
- City name (resolved from lat/lng via Nominatim — see below)

The request does **not** contain:
- Your name or any identifier
- Your device ID
- Your full location history
- Any personal information

The result is cached locally in DataStore for 24 hours. Only one API call is made per day. Anthropic's privacy policy governs their handling of API request data.

---

## 5. Geocoding — Nominatim

City names are resolved from coordinates using [Nominatim](https://nominatim.org), OpenStreetMap's geocoding service. A single request is made at app launch. Only the latitude and longitude are sent. Nominatim's usage policy governs their handling of request data.

---

## 6. Local Storage

All app data is stored locally via Android DataStore. This includes:
- Units preference (°C/°F, km/h/mph)
- Cached daily sentence and its generation date
- Last fetched weather data for offline fallback

Uninstalling the app removes all of this permanently.

---

## 7. Third-Party Services

Nimbus uses three external services:

| Service | Purpose | Data sent |
|---|---|---|
| Open-Meteo | Weather forecasts | Latitude, longitude |
| Anthropic Claude API | Daily sentence generation | Anonymised weather summary |
| Nominatim | City name resolution | Latitude, longitude |

No analytics SDKs. No advertising SDKs. No tracking frameworks. No cloud database.

---

## 8. Children's Privacy

Nimbus does not knowingly collect data from anyone. Since no personal data is collected from any user, no special processing applies to users of any age.
